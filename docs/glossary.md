# Glossary

This glossary defines key terms, acronyms, and concepts used throughout the project documentation.

## A

### Area

A region or part of a town, a country, or the world.

_Source: Oxford Languages via Google, accessed Nov 12, 2025._


## B

### Bunch

A number of things, typically of the same kind, growing or fastened together.

_Source: Oxford Languages via Google, accessed Nov 12, 2025._

## C

### Container

An object that can be used to hold or transport something.

_Source: Oxford Languages via Google, accessed Nov 12, 2025._

## G

### Group

A number of people or things that are located close together or are considered or classed together.

_Source: Oxford Languages via Google, accessed Nov 12, 2025._

## I

### Item

An individual article or unit, especially one that is part of a list, collection, or set.

_Source: Oxford Languages via Google, accessed Nov 12, 2025._

# N

### Node

A point at which lines or pathways intersect or branch; a central or connecting point.

_Source: Oxford Languages via Google, accessed Nov 12, 2025._

## P

### Partition

The action or state of dividing or being divided into parts.

_Source: Oxford Languages via Google, accessed Nov 12, 2025._

# T

### Tree

A thing that has a branching structure resembling that of a tree.

_Source: Oxford Languages via Google, accessed Nov 12, 2025._

## U

### User

A person who uses or operates something, especially a computer or other machine.

_Source: Oxford Languages via Google, accessed Nov 12, 2025._
