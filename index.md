# Fargo

Fargo is a .NET 10 inventory and resource management system. It tracks articles, items, and their placement within a hierarchical partition tree, with real-time event broadcasting and a typed client SDK.

### [Introduction](doc/introduction.md)

Understand the domain model, architecture layers, and core concepts — what Articles, Items, and Partitions are, how authentication works, and how real-time events flow through the system.
